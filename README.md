### Tecnologie

- .NET 10 LTS
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server Management Studio 22
 

### Architettura

Il progetto è strutturato seguendo Clean Architecture, separando chiaramente le responsabilità.

Progetti all'interno della soluzione:

.Domain: Contiene le entità di dominio.

.Application: Contiene la logica di business.

.Infrastructure: Gestisce l'accesso ai dati.

.Api: Espone le API RESTful.
